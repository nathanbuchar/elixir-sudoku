    solve(puzzle)

    Solves the Sudoku puzzle given its string representation.

    Parameters

    • puzzle: The string representation of the Sudoku puzzle.

    Examples 

    iex(1)> puzzle = "0,0,0,0,9,7,0,0,6,...,9,0,0,6,4,0,0,0,0"
